Portfólio - Grupo 4

Seja bem-vindo! Este é um projeto desenvolvido durante as aulas de Desenvolvimento WEB. O site foi projetado pelos estudantes Gabriel de Albuquerque Vasconcelos e Iago Rozales Mahiques do curso de ADS da FIT.

Sobre o Projeto

O projeto foi criado com o intuito de realizar um portfólio totalmente em HTML, buscando deixar o código o mais semântico possível e organizado. O Portfólio de Gabriel contou com tags como <picture> e <source>, ambas utilizadas em conjunto para tornar sua imagem pessoal flexível.

Como o Portfólio foi organizado?

O portfólio, tanto de Gabriel quanto de Iago, foi dividido da seguinte maneira e com a seguinte ordem:

  1. Quem é (Gabriel/Iago)
  2. Experiência
  3. Formação acadêmica
  4. Licenças e Certificados
  5. Idiomas
  6. Redes Sociais
  7. Entre em contato
  8. Navegue pelas páginas

Requisitos mínimos

Foi definido pelo professor, desenvolvedor e mestrando em Engenharia de Software, Caio Eduardo do Prado Ireno, como requisito mínimo para o Portfólio o uso das seguintes tags HTML:

  • <header>
  • <nav>
  • <section>
  • <article>
  • <footer>
  • <ul> ou <ol>
  • <a>
  • <img>
